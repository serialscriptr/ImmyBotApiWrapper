# Pending Computer Count Endpoint
## API Endpoint
> /api/v1/provider-agents/pending-counts
## Description
> Returns a count of the total, offline, failed, manual, recent and active pending computer identifications
## Method
> GET
## Endpoint Parameters
> none/default
## Example output
```sh
total   : 1
offline : 0
failed  : 1
manual  : 0
recent  : 0
active  : 0
```
